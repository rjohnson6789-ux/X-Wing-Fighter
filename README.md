# X-Wing-Fighter
A Star Wars themed Galaga game using python.

Controls:

"A" and "D" move Left and Right, respectivly.
Space is to shoot.

Upgrade:

Blue S icon is Shield Recharge.
Green W icon is Quad Lasers upgrade.
Yellow H icon is Hyperdrive Time Dilation.
Red B Icon is Siesmic Charge Bombs.
